# Japan Terrain Data to minecraft
---

simple little python program to take latitude and longatude cordinates for a region, grab terrain data (5-10m percision), convert it to minecraft cords using the same function as /tpll from [terra++](https://github.com/BuildTheEarth/terraplusplus), and lastly paste it into the world using a small JsMacros helper.

minecraft client needs JsMacros, your client needs /fill /setblock command permissions
